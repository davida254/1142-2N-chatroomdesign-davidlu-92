# AI 輔助使用說明

本專題使用 ChatGPT 作為 AI 輔助工具，使用範圍如下：

1. 分析課程範例聊天室程式不足之處。
2. 協助決定新增功能：多人聊天室、歷史訊息保存、文字截圖快照、管理員禁言、踢人。
3. 協助設計 Server / Client 架構。
4. 協助產生 `multi_server.c` 與 `multi_client.c`。
5. 協助除錯，例如 WSL 環境、gcc 編譯、Segmentation fault、多個 client 訊息不同步等問題。
6. 協助整理 README、流程圖與作業報告。

互動流程：

提供 GitHub 課程範例 → 討論要新增的功能 → AI 提供程式架構與程式碼 → 使用 WSL Debian + VS Code 編譯測試 → 根據錯誤訊息修正 → 製作報告與繳交檔案。
